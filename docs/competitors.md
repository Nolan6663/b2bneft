# Конкурентный анализ ТехЗаказ

> Обновлено: 2026-06-29  
> Позиционирование: **прямые закупки по чертежам/ТЗ без тендера и без комиссии с сделки**

| Тип | Примеры | Суть | Мы конкурируем? |

## Наши клинья (дифференциация)

1. **Закупка по чертежу** — главный бренд, не «ещё одна B2B-площадка»
2. **Загрузка производства** — видна в каталоге и профиле
3. **Горячий матч** — уведомления поставщикам с % и причинами совпадения
4. **Публичный бенчмарк** — на zakupki.html по категории
5. **PDF сравнения КП** — для согласования с руководством
6. **Верификация + отзывы после сделки** — доверие, не просто справочник

## Не копировать

- Тендерный движок (B2B-Center — 20+ лет)
- Каталог SKU (Enex, ПромХаб)
- База 100k+ поставщиков (Supl)
- ЕИС / госзакупки
- Эскроу на старте
