# B2B Platform UI Assets

Внутри:
- `icons_svg/` — иконки в SVG, лучше использовать в Figma/React/Vue.
- `icons_png/` — те же иконки в PNG 64×64.
- `images/ui_showcase_board.png` — общий сгенерированный макет-борд.
- `images/*.svg` и `images/*.png` — простые иллюстрации/empty-state.
- `design_tokens.json` — базовые цвета, радиусы, тени и шрифт.

Важно:
- Иконки сделаны отдельным набором, чтобы не вырезать их из растрового макета.
- Для продакшена лучше вставлять SVG напрямую, а не PNG.
